# classroom web app
## Install dependencies 
#### `npm i` in root directory
#### `cd backend`
#### `npm i` 
## To run react app
#### `npm start` in root directory
## To run express app
#### `cd backend` (in different terminal)
#### `npm run server` 
### Technology used
#### React
#### Node
#### Express
#### MongoDB

### App features
#### Teacher
##### Multiple Team creation
##### Managing team members
##### Assigning tasks/projects
#### Student
##### Teams dashboard
##### Viewing assignments
